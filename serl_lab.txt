## Installation 

### Windows

1. Clone the project on your local machine by typing the following command in CMD.
```bash
git clone https://github.com/Aman1143/serl_lab.git
```

2. Navigate to server  folder and run the following command to download the node_modules for the server.
```bash
npm i
```

3. Start the server by running the following command
```bash
npm start
```

4. Open a new CMD window and navigate to the client folder -> client and run the following commands:
```bash
npm install yarn

yarn add create react-app .
```
```

```bash
yarn add react-router-dom --save
```


```

5. Finally run the following command and a new tab will automatically open in your default browser.
```bash
yarn start
```
